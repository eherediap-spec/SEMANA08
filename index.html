¿Qué es HTML?

HTML es el lenguaje de marcado utilizado para crear la estructura de las páginas web.

HTML

Estructura el contenido de una página web.

CSS

Permite diseñar y personalizar la apariencia.

Bootstrap

Framework que facilita diseños responsivos.

Ventajas de Bootstrap

  • Diseño responsive.
  • Componentes prediseñados.
  • Mayor velocidad de desarrollo.
  • Compatibilidad con dispositivos móviles.
